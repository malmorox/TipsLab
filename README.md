# TipsLab - TFG GSFP DAM 2025-2026

Este proyecto contempla el diseño y desarrollo de una aplicación orientada a la **creación**, **organización** y **consulta** de `life hacks` o consejos prácticos aplicables a la vida cotidiana.

Su alcance se centra en ofrecer una solución funcional y bien estructurada que permita a los usuarios publicar contenido propio, principalmente en formato de vídeo acompañado de una descripción textual, con el objetivo de compartir ideas útiles de forma clara, accesible y visual.

La plataforma permite clasificar los contenidos en distintas categorías (como comida, hogar, organización, tecnología, entre otras), facilitando la navegación y el acceso rápido a la información. Además, incorpora mecanismos de búsqueda y filtrado que ayudan a localizar consejos específicos, guardarlos para su consulta posterior y destacar aquellos más recientes o mejor valorados por la comunidad.
